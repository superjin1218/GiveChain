# GiveChain 🌐

**Plataforma Transparente de Donaciones Globales sobre Stellar**

GiveChain는 Stellar 블록체인 위에 구축된 글로벌 기부 투명성 플랫폼입니다.
모든 기부 내역이 블록체인에 기록되며, 마일스톤 달성 시에만 자금이 방출됩니다.

---

## 🚀 실행 방법 / Quick Start

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:5173` 접속

---

## 🛠 기술 스택 / Tech Stack

- **Frontend**: React 18 + Vite
- **Blockchain**: Stellar (Soroban Smart Contracts)
- **Stablecoin**: USDC
- **API**: Stellar Horizon API

---

## ✨ 주요 기능 / Features

- 실시간 블록체인 기부 추적
- 마일스톤 기반 에스크로 자금 방출
- 수수료 $0.00001 (Stellar 네트워크)
- 기부 실패 시 자동 환불
- 블록체인 익스플로러 통합
